# OpenRender

*A OpenGL Based Render*

## Ref

- JSON reading and analysis: json11
- Model loading: assimp
- Texture Loading: stb
- Math: glm
- Graphics API: OpenGL(GLFW&GLAD)

## Current feature

- RHI
- Loading Model & Texture & Material
- Skybox
- PBR&IBL
- Forward Rendering

## Future......
